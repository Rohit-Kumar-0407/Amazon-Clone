This is the Frontend Clone of one of the famous e-commerce platform, Amazon.
- It is made using HTML and CSS languages
- Tests the complex concepts of CSS such as Flexbox.
- Gave hands-on experience of making frontend of an e-commerce website
- Proud to share that I made this before starting my college.

## Key Features 
1) Mirror Replica:
   Designed in such as way so that it becomes undistinguishable when compared with the original.
2) Easy and Lavish Layout:
   Developed with the help of complex concepts such as Flexbox, Div and Span in CSS, to make it look clear, easy and lavish.
   
## Steps To Run
1) Import Method:
   - Import the whole Github Repository using git or Download the complied rar/zip file.
   - Extract the zip file on local system
   - Click on index.html to open it on local browser.
2) Direct Methods:
   - Click on "" Github Pages deployed link.
   - Click on "more details" and click on "visit unsafe page".
